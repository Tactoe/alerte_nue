ALERTE NUE

Une version google traduite de Deltarune. En partie inspirée par Funky Fantasy. Plutôt recommandé aux personnes ayant déjà fait le jeu qui veulent rire un bon coup.

Avant qu'on pose la question, UnderTale -> DeltaRune -> NudeAlert, donc Alerte Nue.

Installation : Il suffit de DL le zip, extraire le lang_en.json et remplacer l'original situé dans le dossier lang du jeu (Probablement dans les program_files)

Téléchargement

Projet fait pour le fun sur une soirée avec googletrans. Beaucoup de texte est mal aligné, certaines lignes ont une chance de soft-lock le jeu (même si normalement c'est réglé) et il manque les accents et les sprites mais considérez ça comme faisant partie de l'expérience ?? Je compte pas vraiment revenir dessus, vous pouvez faire à peu près ce que vous voulez du json sans nécessairement me mentionner.
